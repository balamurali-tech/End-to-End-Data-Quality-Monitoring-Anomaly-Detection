
# End-to-End Data Quality Monitoring & Anomaly Detection

This project demonstrates a complete data quality workflow using Python and Jupyter Notebook.

## Features
- Missing value analysis
- Duplicate detection
- Data cleaning
- Data visualization
- Outlier analysis
- ML anomaly detection using Isolation Forest
- Data quality scoring
- Clean dataset export

## Tech Stack
- Python
- pandas
- NumPy
- Matplotlib
- scikit-learn
- Jupyter Notebook

## Project Structure

ML_Data_Quality_Project/
├── dataset/
├── output/
├── notebook/
│   └── data_quality_project.ipynb
├── README.md

## Workflow
1. Load Titanic dataset
2. Analyze missing values
3. Remove duplicates
4. Fill null values
5. Visualize distributions
6. Detect anomalies
7. Calculate data quality score
8. Export cleaned dataset

## Dataset
Titanic dataset loaded directly using pandas from public URL.
